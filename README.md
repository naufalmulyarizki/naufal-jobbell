# Naufal Job Bell (FiveM) (QBCore)

Sistem mengirim bell kepada job, dan diterima oleh semua yang job ditentukan dengan kondisi on duty

# Features 
- Mengirim notifikasi panggilan kepada job yang ditentukan
- Mengirim blip panggilan kepada job yang ditentukan, blip lokasinya dia, dan akan hilang setiap 30 menit
- Mengirim suara panggilan kepada job yang ditentukan
- Easy Config

# Installation
- Download Resource
- Masukkan ke dalam resource kamu
- refresh dan start script
